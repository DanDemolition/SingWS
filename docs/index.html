Install On macOS

  1. Open the downloaded DMG.
  2. Drag SingWS into Applications.
  3. Run xattr -dr com.apple.quarantine "/Applications/SingWS.app" once if macOS blocks first launch.
  4. Open SingWS and connect it to your SingWS Server account when needed.
